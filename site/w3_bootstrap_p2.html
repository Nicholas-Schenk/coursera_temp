Progress Bars

to create a default progress bar, add a progress class to a container element and add the progress bar class to its child element.

You can change the height of the progress bar by setting the height attribute in both the progress and progress-bar elements

by writing something in the progress bar div, you can add text to a progress bar such as a numerical percentage of completion

By using any of the bootstrap background colors in the progress-bar you can change the color

The class progress-bar-striped makes them striped

The class progress-bar-animated will make it animated! (only works for striped progress bars)

Progress bars can also be stacked!
